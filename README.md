# Location-Alarm

GPS-based proximity alarm for hiking — alerts you when you approach 
a waypoint within a set distance.

## Two versions available

### 🌐 HTML version (browser-based)
- No installation required
- Works on any device with a browser
- Requires internet connection for map tiles
- [Open in browser](https://asg49.github.io/Location-Alarm/)

### 📱 Android APK
- Works offline (map tiles cached via OSMDroid)
- Background GPS tracking
- Vibration alerts
- Download from [Releases](../../releases)
